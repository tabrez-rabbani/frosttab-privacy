FrostTab — Privacy Policy

Last updated: March 2, 2026

TL;DR FrostTab does NOT collect, store, or transmit any personal data. Everything stays on your device.

What data does FrostTab store?

All preferences are stored locally on your device using Chrome's built-in storage:

  • Theme preference (light/dark)
  • Widget layout positions
  • Widget visibility toggles
  • Clock format (12/24hr)
  • Your display name (for the greeting)
  • Weather cities you've added
  • To-do items
  • Favorite shortcuts
  • Custom wallpaper (stored locally in IndexedDB)

External API Requests

FrostTab requests data from these free, public APIs:

  • Open-Meteo — weather data for your cities (only coordinates sent)
  • GitHub API — public profile stats for your username
  • Google Favicon — website icons for your shortcuts

What we do NOT collect

  • No browsing history
  • No personal information
  • No analytics or tracking
  • No cookies
  • No data sold to third parties
  • No account registration required

Permissions Explained

storage Save your preferences locally
search Chrome search API for search bar
open-meteo.com Fetch weather data
api.github.com Fetch GitHub profile stats

Questions? Reach out via the Chrome Web Store listing.
