🧊 FrostTab — Privacy Policy

Last updated: March 2, 2026

TL;DR: FrostTab does NOT collect, store, or transmit any personal data. Everything stays on your device.

What data does FrostTab store?

FrostTab stores your preferences locally on your device using Chrome's built-in storage APIs:

  • Theme preference (light/dark)
  • Widget layout positions
  • Widget visibility toggles
  • Clock format (12/24hr)
  • Your display name (for the greeting)
  • Weather cities you've added
  • To-do items
  • Favorite shortcuts
  • Custom wallpaper (stored in IndexedDB, on your device)

What data is sent to external servers?

FrostTab makes requests to the following free, public APIs:

  • Open-Meteo API (api.open-meteo.com) — to fetch weather data for cities you've added. Only the city name/coordinates are sent. No personal data.
  • GitHub API (api.github.com) — to fetch public profile stats for the GitHub username you've entered. Only the username is sent. No authentication tokens.
  • Google Favicon Service — to fetch website icons for your favorite shortcuts. Only the domain name is sent.

What data is NOT collected?

  • ❌ No browsing history
  • ❌ No personal information
  • ❌ No analytics or tracking
  • ❌ No cookies
  • ❌ No data sold to third parties
  • ❌ No account registration required

Permissions explained

  • storage — to save your preferences locally
  • search — to use Chrome's search API for the search bar
  • Host permissions — to fetch weather data and GitHub stats from public APIs

Contact

If you have any questions about this privacy policy, you can reach out via the extension's Chrome Web Store listing page.
